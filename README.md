# PortfolioPage

**PortfolioPage** is the personal portfolio website of **TheCanadianYeti**, showcasing professional projects, skills, and achievements. All site files are contained in the `podnarWeb` folder and form the main development hub for the portfolio.

This repository serves as the foundation for TheCanadianYeti's online portfolio, featuring:

* **assets/**: Images, icons, and other media used throughout the portfolio.
